# CH573
CH573 Examples

Create a separate directory for your project.

UnZip the example file into this directory.

double click on the "*.wvproj" file - if the MounRiver IDE has been correctly installed it should
open automatically. 

Connect a WCH-Link USB uploader/programmer to the CH573 as follows:

Link 3V3 to 3V3 pin on the CH573 board....  OR
Link  5V to  5V pin on the CH573 board if you wish to connect ot 5V peripheral devices..

Link GND to Gnd pin on CH573 board.

Link DIO to PB14 on the CH573 Board.
Link CLK to PB15 on the CH573 Board.

Click the Build Project and Download Icon near the top tof the IDE.
The code should be uploaded to the CH573 board and the LED on PA8 should blink continuously.
